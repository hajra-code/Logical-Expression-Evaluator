# Logical-Expression-Evaluator
8086 Assembly logical expression evaluator for bank vault &amp; elevator security. Grants/denies access based on multi-factor Boolean logic (AND/OR/NOT). Built with emu8086.
# Features
- Logical expression evaluator for decision making
- Access control system for bank vault
- Safety condition handling for elevator
- Simulation of real-world scenarios

# Technologies Used
- Assembly Language
- Emu8086

# Learning Outcomes
- Understanding of low-level programming
- Logic design and control flow
- Computer architecture fundamentals
